What is a shortcode? Back to top

A shortcode is a WordPress-specific code that lets you do nifty things with very little effort. Shortcodes can embed files or create objects that would normally require lots of complicated, ugly code in just one line. Shortcode = shortcut.

Slider shortcode Back to top

The shortcode looks like:

[slider id="#"]

Inserting a slider into a Post or Page Back to top

Place the shortcode [slider id="your-slider"] in a post or page to display a slider. Replace "your-slider" with the unique ID of the desired slider.

Inserting a slider into a theme Template file Back to top

Use <?PHP do_action('slider', 'your-slider'); ?> to display a slider anywhere in your template files.
